## 应用

## 如何使用
```
git clone git@github.com:suings/letter.git
```
直接替换media/music.mp3即可替换背景音乐  
主要逻辑在js/index.js中  
主要样式在css/index.css中  
想要更改首页标题文字和次页信件内容,直接在index.html文件中更改即可,样式可以自己定义
